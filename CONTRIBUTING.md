# Contributing to DRAWDD

Thank you for your interest in contributing to DRAWDD! We welcome contributions from the community.

## How to Get Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/drawdd.git`
3. Create a feature branch: `git checkout -b feature/my-feature`
4. Install dependencies: `npm install`
5. Start development: `npm run dev`
6. Make your changes
7. Run linting: `npm run lint`
8. Commit your changes: `git commit -am 'Add my feature'`
9. Push to your fork: `git push origin feature/my-feature`
10. Create a Pull Request

## Development Setup

### Prerequisites
- Node.js 18+
- npm 9+

### Commands
- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run lint` - Run ESLint
- `npm run preview` - Preview production build
- `npm run electron:dev` - Start Electron app in development mode
- `npm run electron:build` - Build Electron installer for Windows

## Code Style

- Follow the existing code style
- Use TypeScript for type safety
- Use Tailwind CSS for styling
- Keep components modular and well-documented

## Reporting Issues

Please report bugs using GitHub Issues with a clear description and steps to reproduce.
